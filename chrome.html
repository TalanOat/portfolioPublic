Summary

This project is an in-progress 2D game developed in Unity. The initial goal is to create a playable demo that establishes the core gameplay loop, art style, and a foundational storyline. Feedback from the demo will guide further improvements, with the ultimate aim of expanding it into a full-fledged game.

Set in a gritty alternate future, the game takes place in a world where humans unknowingly coexist with machines. The player becomes entangled in a gripping mystery that unveils a larger conspiracy. As the main character, players will use their unique abilities to solve puzzles, navigate the city, and uncover the dark criminal underbelly threatening its inhabitants.
